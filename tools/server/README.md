# Server Tools

Local development server bootstrap scripts.

## Scripts

| Script | Purpose |
|--------|---------|
| `launch_server.bat` | Launches a local Rust server with standard configuration |
| `launch_server_progress.ps1` | PowerShell version with progress bar |
| `update_server.bat` | Updates the local server installation |
| `client.connect` | Connection configuration file |

## Usage

```
launch_server.bat
```

Wait for server to fully start, then connect with the game client.
